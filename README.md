# 📄 PDF Editor Pro - Enhanced Signature

**Editor PDF & Gambar dengan Tanda Tangan Digital**  
Aplikasi web ringan, cepat, dan **100% client-side** (tanpa server) untuk menambahkan tanda tangan ke PDF atau gambar dengan mudah.

![Demo](https://via.placeholder.com/800x420/6366f1/ffffff?text=Demo+PDF+Editor+Pro)  
*(ganti dengan screenshot atau GIF demo setelah kamu upload)*

## ✨ Fitur Utama

- Upload **PDF multi-halaman** atau gambar (JPG/PNG)
- Area tanda tangan interaktif (bisa digambar dengan mouse atau touch)
- Import tanda tangan dari file gambar
- **Drag & Resize** tanda tangan di dokumen
- Thumbnail rail + urutkan halaman dengan drag & drop
- Preview real-time di setiap halaman
- Download hasil akhir sebagai **PDF berkualitas tinggi**
- Undo, Clear, Export PNG tanda tangan
- Tampilan modern dengan Tailwind CSS + Plus Jakarta Sans
